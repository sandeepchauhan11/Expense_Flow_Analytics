# FinTrack Dashboard 📊

A lightweight, client-side web application for tracking personal finances. This project was built entirely with Vanilla JavaScript, HTML, and CSS to demonstrate core web development fundamentals without relying on external libraries or frameworks

## 🎯 Project Focus
The primary goal of this project is to showcase real-world frontend business logic, including:
* **State Management:** Handling array manipulations (filtering, reducing) to calculate real-time totals.
* **DOM Manipulation:** Dynamically rendering list items and updating the UI based on user input.
* **Data Persistence:** Utilizing the browser's Native `localStorage` API so user data remains intact after page refreshes.
* **Modern UI Design:** Implementing a clean, responsive interface using CSS Flexbox and semantic HTML.

## ✨ Features
* **Add Transactions:** Users can add income (positive numbers) and expenses (negative numbers).
* **Real-Time Calculations:** The dashboard automatically updates the Total Balance, Income, and Expense cards.
* **Transaction History:** Displays a color-coded list of past transactions.
* **Delete Functionality:** Users can remove incorrect entries, automatically recalculating totals.
* **Persistent Storage:** Data is saved locally in the browser.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and accessible forms.
* **CSS3:** Custom variables, Flexbox layouts, and hover-state animations.
* **JavaScript (ES6+):** Arrow functions, higher-order array methods (`map`, `filter`, `reduce`), and Event Listeners.

## 🚀 How to Run Locally
Because this project uses Vanilla web technologies, no build tools or package managers are required.
1. Clone this repository to your local machine.
2. Open the folder.
3. Double-click the `index.html` file to run it directly in your web browser.
